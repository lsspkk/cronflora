COMPATIBILITY NOTES
===============================================================================

React 19 Breaking Changes:
- Some deprecated APIs removed
- New concurrent features enabled by default
- May require code updates in components using legacy patterns
- Review React 19 upgrade guide: https://react.dev/blog/2024/04/25/react-19-upgrade-guide

Tailwind CSS v4 Migration Required:
- Complete configuration system redesign
- CSS-first configuration approach
- New @theme directive
- @plugin and @config directives
- Upgrade tool available: npx @tailwindcss/upgrade
- Documentation: https://tailwindcss.com/docs

Vite 7:
- Generally backward compatible with Vite 6
- Some plugin API changes
- Better performance out of the box

Node.js 24:
- Improved performance
- Better ES module support
- Enhanced security features
- Latest V8 JavaScript engine
