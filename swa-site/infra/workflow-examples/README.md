# Example Azure Static Web Apps Workflow Files

These are reference examples from official Azure samples for deploying Static Web Apps with APIs.

## Files

1. `sql-bindings-guestbook.yml` - From Azure-Samples/sql-bindings-guestbook
   - Simple deployment with app built by Azure
   - Empty api_location (API auto-discovered)

2. `app-react-vite.yml` - From Azure-Samples/azure-typescript-e2e-apps
   - React + Vite app deployment
   - No API location specified

3. `vanilla-api-example.yml` - Based on staticwebdev/vanilla-api
   - Vanilla HTML/CSS/JS with JavaScript Azure Functions API
   - No build step needed for frontend
